## About Omechu

**오메추(Omechu)** 는 "오늘 뭐 먹지?"라는 매일의 고민을 해결하는 맞춤 메뉴 추천 서비스입니다.
사용자의 취향, 상황, 기분 등 맥락 정보를 기반으로 AI가 개인화된 메뉴와 맛집을 추천합니다.
5단계 온보딩으로 최소한의 입력만으로 바로 나만의 추천을 받아볼 수 있습니다.

<!-- ====================== KEY FEATURES ====================== -->

<details>
<summary><b>Key Features</b></summary>

<br/>

| Feature | Description |
| :--- | :--- |
| **5단계 온보딩** | 최소 입력으로 맞춤 초기 추천 생성 |
| **컨텍스트 기반 탐색** | 태그·가격·정렬·검색 + 무한 스크롤 |
| **AI 메뉴 추천** | OpenAI 임베딩 기반 개인화 추천 |
| **맛집 상세** | 리뷰·평점·사진을 한눈에 |
| **찜 & 추천 목록** | 낙관적 업데이트로 즉시 반응 UX |
| **마이페이지** | 프로필 / 기본 상태 / FoodieLog |
| **메뉴배틀** | 두 메뉴 중 하나 선택하는 재미 요소 |

<br/>

</details>
